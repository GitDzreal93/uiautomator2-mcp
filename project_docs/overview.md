# UIAutomator2 MCP Server

## 项目概述

这是一个基于FastMCP框架实现的UIAutomator2 MCP服务器,它将UIAutomator2的Android自动化测试功能通过MCP协议提供给Claude等AI助手使用。

## 核心功能

- 提供Android设备控制能力
- 实现基础的UI自动化操作
- 支持截图和状态获取
- 提供应用管理功能

## 技术栈

- Python 3.10+
- FastMCP
- UIAutomator2
- ADB工具

## 目标用户

- 需要通过AI助手控制Android设备的开发者
- 需要自动化测试Android应用的测试人员
- 想要通过自然语言控制Android设备的用户
